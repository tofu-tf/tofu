tofu.doobie.transactor

Txr

Companion object Txr

trait Txr[F[_], DB0[_]] extends AnyRef

A simple facade for doobie.Transactor that holds an inner database effect type DB[_] and provides natural transformations from this effect to the target effect F[_].

The motivation for using this facade instead of Transactor is to:

  • initialize all its natural transformations early and remove the need for additional constraints on F[_] later (e.g. Bracket);
  • be able to use another DB[_] effect besides ConnectionIO and build a layer of transactional logic with it.
